001 - BANANA Add-in

This sample shows how to extract some data from Banana document and load it into the worksheet using the Excel JavaScript API.

1) Update the list with the current opened Banana documents

2) Select a Banana document file

3) Select a table (*)

4) Generate excel file

(*)

- Tables should be added on different sheets.

- It is possible to overwrite an existing table: be sure to select the right sheet.

- It is not possible to create a new table if it already exists.
